# 🚀 ByteCraft Month 1 Project

## 📌 Project Overview

This project was developed during the first month of my internship at **ByteCraft IT Solutions**.
It is a **Company Portfolio & Client Management Web Application** built using frontend technologies.

The main objective of this project is to demonstrate:

* Basic web development skills
* UI design and responsiveness
* JavaScript functionality and DOM manipulation

---

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript (DOM Manipulation)
* Bootstrap 5
* Browser LocalStorage

---

## ✨ Features

### 🔹 Home Page

* Project overview dashboard
* Navigation to different modules
* Displays total number of clients dynamically

### 🔹 Client Management Module

* ➕ Add new clients
* ✏️ Edit/update client details
* ❌ Delete clients with confirmation
* 🔍 Search/filter clients
* 📊 Display total client count
* 💾 Data persistence using LocalStorage

### 🔹 Contact Module

* 📩 Contact form with validation
* 🧾 Stores submitted messages
* 📜 Displays previously sent messages
* 💾 Data stored using LocalStorage

## ▶️ How to Run the Project

1. Download or clone the repository
2. Extract the project folder
3. Open `index.html` in any browser

OR

Use VS Code with Live Server:

* Right-click `index.html`
* Click **Open with Live Server**
